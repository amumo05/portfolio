Tools: Python, Matplotlib, Structured Data Definitions

Overview

This project analyzes the financial statements of major companies to understand long-term profitability trends. The goal was to visualize how net profit margins evolve over time and identify patterns of growth, stability, or decline using structured data and clear visual representations.

Visuals

Detailed Explanation

The dataset consists of financial records for multiple companies across several years. Each record includes the year, company name, sector, market capitalization, revenue, and net profit margin. These structured fields allow the program to perform meaningful time-series analysis.

Using Python, the data is processed through a main analysis function that computes net profit margins and applies a three-year moving average to smooth short-term fluctuations. The results are then visualized using Matplotlib to highlight overall profitability trends and long-term growth behavior.
